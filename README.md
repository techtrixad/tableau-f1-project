\# 🏎️ F1 Race Intelligence Dashboard



\## 📌 Overview

An interactive Tableau dashboard analyzing Formula 1 racing data

across multiple seasons (2021-2023), providing insights on race 

pace, pit strategies, driver performance and weather impact.

\## 📊 Dashboard Features

\- 🏁 \*\*Race Pace Line Chart\*\* — Avg lap times per driver per lap

\- 🔧 \*\*Pit Stop Gantt Chart\*\* — Pit stop strategies visualization

\- 👑 \*\*Driver of the Day\*\* — Performance scoring across races

\- 🌦️ \*\*Weather Impact Analysis\*\* — Circuit wise weather risk

\- 🏆 \*\*Constructor Performance\*\* — Team points across seasons

\- 🏎️ \*\*Qualifying vs Race Performance\*\* — Grid vs finish analysis
